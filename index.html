Built for speed and privacy

Everything you need for seamless internal team communication without relying on external servers.

Instant Messaging

Real-time WebSockets ensure messages are delivered instantly across all connected devices.

Unlimited File Size

Bypass email and cloud limits. Chunked uploads handle multi-gigabyte files flawlessly.

Fully Private

Your data never leaves your router. Air-gapped security by default. No analytics, no tracking.

Single Device Setup

Only the host runs the executable. Clients simply open a URL in any modern browser.

Cross-Platform

Works natively on macOS, Windows, Linux, iOS, and Android via the browser client.

Full Control

Senders retain the ability to permanently delete shared files from the local server.
